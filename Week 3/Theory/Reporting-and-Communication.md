# Reporting and Stakeholder Communication

Effective reporting is essential to translate technical risks into business impact.

## Report Structure
- Executive Summary
- Technical Findings
- Impact Analysis
- Remediation

## Audience Tailoring
Technical details were documented for security teams,
while a non-technical summary was created for management.

## Metrics Used
- Severity: Critical
- Exploit success rate: 100%
- Time to compromise: < 5 minutes
